# Go Server

A simple Go server project.

## Features

- Lightweight and fast
- Easy to configure
- RESTful API support

## Getting Started

### Prerequisites

- [Go](https://golang.org/dl/) 1.18 or higher

### Installation

```bash
git clone https://github.com/yourusername/go_server.git
cd go_server
go mod tidy
```

### Running the Server

```bash
go run main.go
```

The server will start on `http://localhost:8080`.
